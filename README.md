# 城崎温泉・天橋立 しおり

## ファイル構成
- `shiori.html`: 配布用の本体（スマホ閲覧・印刷対応）
- `plan.json`: 行程・ルール・出典の編集用データ
- `index.html`: `shiori.html` へのリダイレクト

## ブラウザで開く方法
1. エクスプローラーで `shiori.html` をダブルクリック
2. もしくはブラウザへドラッグ&ドロップ

## 印刷手順（2〜3枚想定）
1. `shiori.html` を開く
2. 右上の `🖨️` ボタンを押す（または `Ctrl+P`）
3. 印刷設定で以下を推奨
- 余白: 標準
- 背景グラフィック: ON（好み）
- 用紙: A4

## iPhone で共有する方法
1. `shiori.html` を LINE にファイル送信
2. または Google Drive / iCloud Drive にアップロードして共有リンク送付
3. 受信側は Safari で開き、必要なら「共有 > プリント」で印刷

## 運用メモ
- 営業時間・料金は変動するため、前日夜に `shiori.html` 内の公式リンクを再確認
- 特に Day2 の固定時刻
  - `15:10` 天橋立エリア出発
  - `20:00` 松井山手で祖母送迎
